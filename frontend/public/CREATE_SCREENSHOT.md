# How to Create Dashboard Screenshot

## Option 1: Use the Preview HTML (Recommended)

1. **Open the preview file:**
   - Navigate to: `frontend/public/dashboard-preview.html`
   - Double-click to open in your browser

2. **Take a screenshot:**
   - Press `F11` to go fullscreen (optional)
   - Press `Windows + Shift + S` to open Snipping Tool
   - Select the dashboard area
   - Save as `dashboard-screenshot.png` in the `frontend/public/` folder

## Option 2: Screenshot from Live Dashboard

1. **Open your live dashboard:**
   - Go to: `agent.liquidcanvas.art/dashboard`
   - Make sure you're logged in

2. **Take a screenshot:**
   - Press `Windows + Shift + S` (Windows Snipping Tool)
   - Or use browser DevTools: `F12` → Click device toolbar → Take screenshot
   - Save as `dashboard-screenshot.png` in `frontend/public/`

## File Location:
```
frontend/
└── public/
    └── dashboard-screenshot.png  ← Save here
```

Once saved, the landing page will automatically display it!
