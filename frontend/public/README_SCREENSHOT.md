# Dashboard Screenshot

To add the dashboard screenshot to the landing page:

1. **Save the screenshot** you provided as `dashboard-screenshot.png`
2. **Place it in this directory** (`frontend/public/dashboard-screenshot.png`)
3. The landing page will automatically display it

**Recommended specifications:**
- Format: PNG (for best quality)
- Aspect ratio: 16:9 (e.g., 1920x1080, 1920x1200)
- File size: Keep under 2MB for fast loading

The landing page is already configured to display this image. Once you save it here, it will appear automatically!
