# How to Add Dashboard Screenshot to Landing Page

## Quick Steps:

1. **Take a screenshot** of your dashboard at `agent.liquidcanvas.art/dashboard`
   - Press `Windows + Shift + S` to take a screenshot
   - Or use any screenshot tool

2. **Save the screenshot** as `dashboard-screenshot.png`
   - Save it to: `C:\Users\MIKENZY\Documents\Apps\liquidcanvas\frontend\public\dashboard-screenshot.png`

3. **Verify the file exists:**
   - The file should be at: `frontend/public/dashboard-screenshot.png`
   - File name must be exactly: `dashboard-screenshot.png`

4. **The landing page will automatically display it!**
   - Visit `agent.liquidcanvas.art` to see it

## File Path:
```
liquidcanvas/
└── frontend/
    └── public/
        └── dashboard-screenshot.png  ← Save your screenshot here
```

## Recommended Screenshot Settings:
- **Format:** PNG (best quality)
- **Size:** 1920x1080 or similar (16:9 aspect ratio)
- **Content:** Capture the main dashboard area showing stats cards and controls

Once saved, refresh the landing page and the screenshot will appear!
